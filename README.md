Android App Web Migration Project
Overview

This repository contains the React-based web version of the existing Android application. It is designed to provide feature parity while introducing desktop-optimized workflows.

Features

Responsive Navigation: Adaptive header for desktop and bottom-bar for mobile viewports.

Shared API Interface: Uses the same backend logic as the mobile app.

Offline Support: Basic PWA capabilities for improved performance on slow networks.

Setup

Clone the repo.

Run npm install.

Create a .env file with VITE_API_BASE_URL.

Run npm run dev to launch.
